Tool for e2e & visual regression

Installation

Clone the repo and go in the root folder, then:
npm install
For technical tests, PhantomJS is used, with a standalone Selenium server (available here: http://selenium-release.storage.googleapis.com/index.html)
java -jar selenium-server-standalone-X.XX.X.jar
For e2e tests, you can use PhantomJS or Browserstack.

To use PhantomJS, set this value:
export HEADLESS_PHANTOM=true
If you want to use Browsterstack, configure your informations: (available here: https://www.browserstack.com/accounts/settings)
export BROWSERSTACK_USERNAME=YOUR_USERNAME
export BROWSERSTACK_ACCESS_KEY=YOUR_KEY
export HEADLESS_PHANTOM=false
And run:
npm run test
If visual diff fails, you'll have to review each screenshot in screenshots/diff & screenshots/current and validate that it's ok. If it's ok, run:
npm run test:validate

Configuration

Change the tested URLs: go in the config/pages.js file and edit the baseUrl property, and the list of URLs to test.
Change the browser list: Go in the config/browsers.js file and edit the browsers property (for Browserstack only).

Tests done

Technical

  • title & description not empty
  • Only one <h1>
  • No heading is hidden
  • No heading is empty
  • No break in the heading hierarchy
  • No W3C error

Visual regressions

  • No visual diff with the previous validated screenshot
