Tool for e2e & visual regression tester

Installation

Clone the repo and go in the root folder, then:
npm install
Configure your Browserstack informations: (available here: https://www.browserstack.com/accounts/settings)
export BROWSERSTACK_USERNAME=YOUR_USERNAME
export BROWSERSTACK_ACCESS_KEY==YOUR_KEY
And run:
npm run test
If visual diff fails, you'll have to review each screenshot in screenshots/diff & screenshots/current and validate that it's ok. If it's ok, run:
npm run test:validate

Configuration

Change the tested URL: go in the wdio.conf.js and edit the baseUrl property.
Change the browser list: Go in the wdio.conf.js and edit the capabilities property.
